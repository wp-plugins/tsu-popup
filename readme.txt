== Description ==
 Your visitors can easily visit your Tsu profile and use it as your invitation to the network. You can optionally add a name to be displayed and choose different logo image.

== Installation ==

= Using The WordPress Dashboard =

1. Navigate to the 'Add New' in the plugins dashboard
2. Search for 'Tsu Popup'
3. Click 'Install Now'
4. Activate the plugin on the Plugins settings page

= Uploading in WordPress Dashboard =

 At first upload the zip file tsu-popup form your wrodpress admin panel. You may upload the plugin files by using FTP too.
<a href="https://codex.wordpress.org/Managing_Plugins">Here have details guidelines to install a wordpress plugin</a>

After install Tsu Popup plugin, Activate that and go to the settings  page for setting your Popup.

1. Upload `the plugin in 'wp-content/plugins/' directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Now use your plugin
